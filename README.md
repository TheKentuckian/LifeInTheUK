# Life in the UK Test - Practice Website

A mobile-friendly practice website for the Life in the UK test, helping you prepare for British citizenship or settlement.

## Features

- **Authentic Test Format**: 24 questions, 45-minute timer, 75% pass mark (18/24 correct)
- **100+ Practice Questions**: Comprehensive question bank covering all 5 chapters of the official handbook
- **Mobile-Optimized**: Responsive design that works perfectly on smartphones
- **Instant Results**: Detailed feedback showing which questions you got right and wrong
- **Progress Tracking**: Visual progress bar and question navigation
- **Random Selection**: Each test randomly selects 24 questions for varied practice

## Test Chapters Covered

1. **The Values and Principles of the UK** - Fundamental British values and rights
2. **What is the UK?** - Geography, countries, and symbols
3. **A Long and Illustrious History** - British history from Stone Age to modern times
4. **A Modern, Thriving Society** - Culture, sports, arts, and traditions
5. **The UK Government, the Law and Your Role** - Government structure, legal system, and citizenship

## How to Use

Simply open `index.html` in any web browser to start practicing. The website works offline and requires no installation.
